Tutorial — Classificador NCM

Aprenda a usar o Classificador NCM para identificar o código fiscal correto de qualquer mercadoria, com embasamento nas Regras Gerais de Interpretação (RGIs) e na NESH.

O que é o NCM?

O NCM (Nomenclatura Comum do Mercosul) é um código de 8 dígitos usado para classificar mercadorias no comércio exterior brasileiro. Ele determina quais impostos (II, IPI, PIS, COFINS) incidem sobre um produto e se ele precisa de licenças especiais para importação ou exportação.

Classificar errado pode gerar multas, apreensão da mercadoria e autuações da Receita Federal. O Classificador usa IA para sugerir o código correto com base nas RGIs — as mesmas regras usadas pelos despachantes aduaneiros.

Como usar o Classificador NCM

1
Acesse a página do Classificador
Clique em Identificar NCM no menu lateral. A página abrirá com um formulário de consulta.
2
Preencha o nome do produto
No campo Nome / descrição do produto, descreva o produto com o máximo de detalhes possível. Quanto mais específico, melhor o resultado.

Exemplos de boas descrições:
Chapa de aço carbono 1008 galvanizada a fogo, espessura 1,52mm
Notebook com processador Intel Core i7, 16GB RAM, SSD 512GB
Luva de borracha nitrílica descartável para uso industrial, tamanho M
3
Adicione informações técnicas (opcional, mas recomendado)
No campo de informações adicionais, inclua dados como:

CampoO que informar
MaterialComposição, liga metálica, tipo de polímero
DimensõesEspessura, largura, comprimento, peso
UsoFinalidade do produto, setor onde é usado
ProcessoComo foi fabricado (trefilado, laminado, injetado...)
4
Anexe imagem ou documento (opcional)
Você pode arrastar ou selecionar uma imagem (JPG, PNG, WEBP) ou PDF do produto. O sistema usará essas informações para melhorar a classificação.
Nota: O modelo Groq atual (llama-3.3-70b) não processa imagens diretamente. Use o campo de texto para descrever o que está na imagem quando precisar de maior precisão.
5
Clique em "Identificar NCM"
O sistema enviará os dados para a IA e retornará o resultado em alguns segundos.

Interpretando os resultados

O resultado é dividido em três abas:

1
Sugestões NCM
Exibe o código principal recomendado com nível de confiança:
Alta — código bem definido Média — verificar alternativas Baixa — consultar despachante
Também mostra alternativas com a explicação de quando cada uma se aplica.
2
Embasamento
Mostra o percurso classificatório (Seção → Capítulo → Posição → NCM) e quais Regras Gerais de Interpretação (RGIs) foram aplicadas, com referência ao Decreto nº 11.889/2023.
3
Minuta de Laudo
Gera um laudo técnico formal que pode ser usado como resposta a questionamentos da Receita Federal. Clique em Gerar minuta de laudo e depois baixe o arquivo .txt.
⚠ Atenção legal: A sugestão do classificador é um ponto de partida. A classificação definitiva é responsabilidade do importador/exportador. Em casos complexos ou de alto valor, consulte sempre um despachante aduaneiro habilitado.

Como obter uma chave da API Groq (gratuito)

O Classificador NCM usa a API da Groq para processar as consultas com IA. A Groq oferece um plano gratuito generoso — sem necessidade de cartão de crédito para começar.
1
Crie uma conta no console da Groq
Acesse o link abaixo e clique em Sign Up. Você pode entrar com Google ou GitHub.
🔑
console.groq.com/keys
Console oficial da Groq — crie e gerencie suas chaves de API
 ↗
2
Gere uma nova chave
Após fazer login, clique em Create API Key, dê um nome (ex: thiago-tools) e copie a chave gerada. A chave começa com gsk_.
Importante: Copie a chave imediatamente — ela não será exibida novamente depois que você fechar a janela.
3
Salve a chave nas Configurações do sistema
Acesse Configurações no menu lateral, cole sua chave no campo Chave da API Groq e clique em Salvar. A partir daí o Classificador NCM usará a sua chave.
